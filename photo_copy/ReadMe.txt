//　photo copy for windows 

// 概要
photo_copyはWindowsで動作するjpegとRAWファイルをNAS等にcopyすることを目的に作られたコンソールアプリケーションです。
jpeg以外のRAWファイル名は各社違いますが、標準ではrw2とorfに対応しています。
追加するには引数でextで複数のファイル名を渡せます。
また、コピー先は日付ごとにフォルダを作成してそこにコピーされます。

// 使い方
C:\ photo_copy.exe --src d:\ --dst \\10.10.10.10\backup_photo
上記サンプルではd:\から再帰的に適応する拡張子のファイルを検索し、\\10.10.10.10\backup_photo以下にファイルの更新時間でディレクトリを作成し、そこにコピーします。

C:\ photo_copy.exe --src d:\ --dst \\10.10.10.10\backup_photo --ext .ps2,.tiff
上記サンプルではd:\から再帰的に適応する拡張子とps2とtiffの拡張子のファイルを検索し、\\10.10.10.10\backup_photo以下にファイルの更新時間でディレクトリを作成し、そこにコピーします。
